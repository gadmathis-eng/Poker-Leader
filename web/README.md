# Pot Master web pages

Public site for **https://potmaster.app**

| Page | App Store field | URL |
|------|-----------------|-----|
| Privacy | Privacy Policy URL | https://potmaster.app/privacy/ |
| Support | Support URL | https://potmaster.app/support/ |
| Home | Optional | https://potmaster.app/ |

## Deploy

See **[DEPLOY-potmaster.app.md](./DEPLOY-potmaster.app.md)** for Vercel, Cloudflare Pages, Netlify, or GitHub Pages.

## Local preview

```bash
cd web
python3 -m http.server 8080
```

Open http://localhost:8080/privacy/ and http://localhost:8080/support/
